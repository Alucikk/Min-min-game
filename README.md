# Min min 🐾

**Min min** — це динамічна браузерна гра у жанрі аркади, написана на чистому JavaScript з використанням Canvas API. Гравець керує чорним міні котиком, який має долати перешкоди, підлаштовуючись під різні рівні складності.

## 🎮 Особливості проєкту
- **Об'єктно-орієнтоване програмування (ООП):** Логіка гри розділена на класи (`Game`, `Cat`, `Pipe`, `UIGame`), що полегшує підтримку та розширення коду.
- **Система профілів:** Можливість ввести ім'я гравця для збереження персональних рекордів.
- **Рівні складності:** Три режими - легкий, середній, складний, які змінюють швидкість гри та частоту появи перешкод.
- **Адаптивність:** Використання `deltaTime` та `timeStep` забезпечує однакову швидкість гри на моніторах з різною частотою оновлення (60Hz, 144Hz тощо).
- **Стилістика:** Авторський дизайн у фіолетово-синіх тонах з використанням класичних шрифтів.

## Технології
- **HTML5** (Canvas)
- **CSS3** (Flexbox, Animations)
- **JavaScript** (ES6+, OOP)

## Інструкції з написання та підтримки коду

Для збереження чистоти та стабільності коду при подальшій розробці, дотримуйтесь наступних правил:

### 1. Архітектура класів
Кожен елемент гри повинен бути інкапсульований у свій клас:
- `Cat`: відповідає лише за фізику стрибка та рендеринг персонажа.
- `Pipe`: описує логіку руху та малювання перешкод.
- `UIGame`: керує відображенням тексту на канвасі та станами екранів (Game Over, Start).
- `Game`: головний контролер (Engine), який об'єднує всі класи та керує ігровим циклом.
### 2. Принципи оновлення кадрів
Не використовуйте фіксовані значення для руху об'єктів. Завжди використовуйте `deltaTime`:
```javascript
update(deltaTime) {
    let timeStep = Math.min(deltaTime, 100) / 16.6;
    this.x -= this.speed * timeStep;
}
